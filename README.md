## Features

- Multiple quiz formats with varying difficulty levels
- Gamification elements such as streaks, rewards, and trophies
- User authentication and profile management
- Backend integration with REST APIs and database storage
- Responsive and interactive mobile interface built with React Native

## Tech Stack

**Frontend & Mobile:** React Native (Expo), TypeScript  
**Backend & Database:** Node.js, Express, MongoDB  
**Testing & Tools:** Git, GitHub, REST API integration
